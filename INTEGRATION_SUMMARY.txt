API INTEGRATION PATTERN
================================================================================

All pages follow this pattern:

1. Import API service
   import { citizenAPI, collectorAPI, adminAPI } from '@/services/api'

2. Add state management
   const [data, setData] = useState([])
   const [isLoading, setIsLoading] = useState(true)
   const [error, setError] = useState(null)

3. Load data on mount
   useEffect(() => { loadData() }, [])
   
   const loadData = async () => {
     try {
       const response = await citizenAPI.getPickupRequests()
       setData(response.data?.items || [])
     } catch(err) { setError(err.message) }
     finally { setIsLoading(false) }
   }

4. Render with states
   if (isLoading) return <Spinner />
   if (error) return <ErrorMessage />
   return <DataDisplay data={data} />

TROUBLESHOOTING
================================================================================

ISSUE: "Failed to load data"
FIX: Make sure backend server is running on http://localhost:5000

ISSUE: "401 Unauthorized"
FIX: Token may have expired. Clear localStorage and login again.

ISSUE: "Cannot read property 'items'"
FIX: Backend response format may vary. Use: response.data?.items || []

ISSUE: Blank page with spinner
FIX: Check Network tab for failed API calls. Check browser console logs.

ISSUE: CORS errors
FIX: Make sure backend has CORS enabled for localhost:5173

ISSUE: Data not updating
FIX: Check that backend has fresh data. Try refreshing the page.
